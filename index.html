Barcharts

Stacked bar-charts for number of messages vs time.

Download this project as a .zip file Download this project as a tar.gz file

About.

This is an attempt to get a good insight at d3.js. I have used messages data from one of the applications I am currently working on. Users interact by sending each other messages to form teams. One of the latest runs of 'My Dream Team' was scheduled from 19th Feb to 23rd Feb, 2014. The stacked bar-charts show the count of messages per day and per hr as the deadline approaches.
